# JavaScript Command-Line Project

This project includes three Node.js command-line programs:

1. Rock Paper Scissors game
2. Pig Latin translator
3. Caesar Cipher encryption tool

## How to Run

Use Node.js to run each program:

node rock-paper-scissors.js rock
node pig-latin.js "Hello world"
node caesar-cipher.js "hello world" 3
